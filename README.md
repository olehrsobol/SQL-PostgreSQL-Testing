# SQL-PostgreSQL-Testing

This repository contains SQL queries and database testing examples using the <strong><a href="https://github.com/xzilla/pagila">Pagila database</a></strong> (PostgreSQL sample DB).

The goal was to practice SQL querying and validate data integrity using real-world-like database structure.
